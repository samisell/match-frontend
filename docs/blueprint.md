# **App Name**: HeartCraft

## Core Features:

- Profile Submission: Multi-step form for users to submit detailed profiles with personal information, lifestyle, preferences, and photos.
- Manual Matching Display: Display of matched profiles in a grid with key information and an option to view a detailed profile summary with admin notes (UI only).
- User Dashboard: A personalized dashboard displaying profile status, number of matches, profile summary, and admin updates (UI only).
- AI-Powered Profile Analyzer: Leverage an AI tool to suggest profile improvements based on successful match criteria, offering guidance on highlighting key attributes and interests to attract compatible matches.
- Contact Form with Validation: A contact form page with complete validation and clear prompts, where users can send inquiries to support and receive a message accordingly.
- Matches Carousel: Once profiles are created, the profile information is visible on the landing page in a carousel for increased user engagment

## Style Guidelines:

- Primary color: Dusty Rose (#D8ADA4) to evoke romance and trust.
- Background color: Soft, desaturated beige (#F5F5DC) for a clean, comforting feel.
- Accent color: Muted lavender (#B2A8B5) to provide contrast and highlight key actions without being overly assertive.
- Font: 'Alegreya', a humanist serif font suitable for headlines or body text, for a romantic and sophisticated feel.
- Use subtle, elegant icons to represent different aspects of the platform, like hearts for matches and speech bubbles for messages.
- Card-based layouts with rounded corners and soft shadows to provide a modern, approachable design.
- Smooth hover and transition effects to enhance user interaction and create a sense of polish.
